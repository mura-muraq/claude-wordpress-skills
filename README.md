# MuRa's Claude Code WordPress Skills

Claude Code plugins and skills for WordPress by MuRa.

## Plugins and Skills

| Plugins and Skills | Description |
|--------|-------------|
| [wp-jwt-rest-api](plugins/wp-jwt-rest-api/skills/wp-jwt-rest-api/SKILL.md) | A CLI bash script for calling the WordPress REST API with JWT (JSON Web Token) authentication. Automatically authenticates via JWT before running curl. Works behind HTTP Basic Auth as well. |

## Installation

**1. Launch Claude Code**
```bash
claude
```

**2. Add the marketplace**

```
/plugin marketplace add mura-muraq/claude-wordpress-skills
```

**3. Install plugins**

```
/plugin install wp-jwt-rest-api@mura-wordpress
```
